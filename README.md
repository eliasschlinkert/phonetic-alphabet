# 📞 Phonetic Alphabet Tool

A simple, lightweight browser-based tool to assist with the German and the NATO phonetic alphabet. <br> Quickly look up the phonetic equivalent of any letter — no installation, no dependencies, just open and use. <br> When spelling out words over the phone or radio, misunderstandings are common. The NATO phonetic alphabet assigns a unique word to each letter (e.g. A = Alpha, B = Bravo) to avoid confusion. This tool helps you quickly reference or convert text to its phonetic equivalent.

Works for both desktop and mobile browsers.

---

## Usage

Clone the repository and open `index.html` in any modern browser:

```bash
git clone https://github.com/eliasschlinkert/phonetic-alphabet.git
```

No further installation or dependencies required.

If necessary, a local web server can also be set up.
